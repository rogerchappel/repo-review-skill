# Contributing

Thanks for helping improve `repo-review-skill`.

## Development

Use Node.js 18 or newer.

```bash
npm install
npm run release:check
```

## Pull Requests

- Keep changes focused and include tests or fixture updates for behavior changes.
- Run `npm run release:check` before opening a PR.
- Avoid committing generated review output unless it is a deliberate fixture.

## Issues

Please include the command you ran, the repository shape you tested against, and the expected versus actual output.
